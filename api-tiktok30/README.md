# 🎂 Happy 30th Birthday — Adee Uti

Sebuah halaman web perayaan ulang tahun yang penuh cinta dan rasa syukur.

## 📁 Struktur Project

```
birthday-adee/
├── index.html          ← Halaman utama
├── style.css           ← Styling (Ivory × Rose × Gold)
├── script.js           ← Animasi, musik, interaksi
└── assets/
    └── semua-aku-dirayakan.mp3   ← Masukkan file lagu di sini
```

## 🎵 Cara Menambahkan Lagu

Letakkan file MP3 di:
```
assets/semua-aku-dirayakan.mp3
```

Lagu akan otomatis diputar saat pengunjung pertama kali berinteraksi dengan halaman.

## 🌸 Fitur

- **5 Scene** yang mengalir dengan scroll
- **Partikel emas** mengambang di latar
- **Kelopak bunga** melayang-layang
- **Timeline kenangan** dengan animasi scroll
- **Tombol rayakan** dengan efek confetti & heart burst
- **Musik latar** yang bisa di-toggle
- Font elegan: Playfair Display + Cormorant Garamond

## 🚀 Cara Buka

Cukup buka `index.html` di browser — tidak perlu server!
